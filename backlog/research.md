## MemoryArena — Multi-Session Agentic Memory Benchmark
- **URL:** https://arxiv.org/abs/2602.16313
- **Authors:** Zexue He, Yejin Choi, Alex Pentland et al. (UCSD/UW/MIT)
- **Published:** 2026-02-18
- **Why:** Benchmarks memory in Memory-Agent-Environment loops — agents must learn from earlier actions and use memory to guide later ones. Shows that systems saturating LoCoMo/LongMemEval fail in agentic settings. HARDER and more realistic than LongMemEval.
- **Categories:** Web navigation, preference-constrained planning, progressive info search, sequential reasoning
- **Key finding:** Memorization and action are tightly coupled. Testing them in isolation misses the real challenge.
- **Priority:** HIGH — this is the benchmark we should target after LongMemEval. Winning here = real dominance.
- **Added:** 2026-02-19
